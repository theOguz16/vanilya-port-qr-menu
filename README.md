# Vanilya Port QR Menü

Vanilya Port için mobil uyumlu QR menü ve basit admin paneli.

## Çalıştırma

```bash
python3 server.py --host 0.0.0.0 --port 4948
```

Menü:

```text
http://localhost:4948/
```

Admin:

```text
http://localhost:4948/admin/
```

Admin giriş:

```text
Kullanıcı adı: VanilyaPort
Şifre: Vanilya2017Port
```
